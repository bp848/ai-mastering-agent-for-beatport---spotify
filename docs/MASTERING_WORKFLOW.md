# Beatport 納品のマスタリングワークフロー

プロ向けの推奨手順です。

| 手順 | 内容 |
|------|------|
| ① マスタ作成 | 24 bit / 48 kHz（または 96 kHz）でマスタリングし、ヘッドルームを確保する。 |
| ② ハイレゾ保存 | 完成した 24 bit マスタは RAW または WAV で保存し、バックアップする。 |
| ③ 配信用ダウンコンバート | 同じマスタから 16 bit / 44.1 kHz に変換する（**Dither 付き推奨**）。 |
| ④ Beatport 納品 | 16 bit / 44.1 kHz WAV を Beatport（またはレーベル経由）に提出する。 |
| ⑤ アーカイブ | 24 bit マスタは別途保管し、将来のハイレゾ配信やリマスタリングに備える。 |

## Tip: Dither（量子化ノイズの最小化）

**Dither**（特に **TPDF = Triangular PDF**）をかけて 24 bit → 16 bit に変換すると、量子化ノイズが最小化され、最終的な音質が向上します。

- ビット深度を下げる際に必ずしも「切り捨て」ではなく、ノイズシェーピング＋dither を適用するのがプロの一般的なやり方です。
- 本アプリで配布用 WAV を 16 bit / 44.1 kHz で出力する場合も、内部で 24 bit マスタを経由し、ダウンコンバート時に dither を適用する実装が望ましいです。
